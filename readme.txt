A powerful Facebook and Twitter integration that allows you to display facebook, twitter follow button and timeline for your wordpress website.

== Description ==
<p>A powerful Facebook and Twitter integration that allows you to display facebook (timeline, events and messages), twitter follow button and twitter timeline for your wordpress website.</p>
<p>This plugin is very useful for displaying facebook page like button, twitter follow button and feeds. It will improve page likes and followers.</p>
<p>Just activate and setup your facebook and twitter pages. It will display on your website at left/ right side of the window.</p>

<h4>Features</h4>
<pre>
<code>
* Easy to Gain more Facebook page likes and twitter followers
* Display facebook timeline, events and messages
* Twitter feeds inegration
* Add/Remove Twitter follow button with its customization
* Very simple to setup & maintain
* Clean & simple plugin options page
</code>
</pre>

== Installation ==
1. Put the folder named "facebook-twitter-feeds" under '/wp-content/plugins/' directory or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress. For more details, http://codex.wordpress.org/Managing_Plugins
3. Goto Settings -> FBTW Feeds.
4. Add facebook and twitter page information. The feeds will be showing at your website.
